To set game settings open command line inside game folder and enter:
game.exe -window "WIDTHxHEIGHT" -map "WIDTHxHEIGHT" -num_enemies "amount" -num_ammo "amount"

Example:
game.exe -window 800x600 -map 1000x1000 -num_enemies 10 -num_ammo 3

CONTROLS:
move: keyboard arrows
shoot: mouse right click
start/pause/unpause/proceed: mouse left click
suicide: wheel click

Amount of enemies increases after each victory and decreases after every death (resets to 10 if less then 1)
Game boundaries signaled by 4 stars in each corner of gamefield
